# 我用AI从零写了一个股票分析App，完全离线运行，开源免费

## 为什么做这个？

作为一个散户，我经常遇到这些问题：

- 同花顺广告太多，核心功能要付费
- 雪球内容质量参差，信息过载
- 想要回测自己的策略，但现有工具门槛太高
- 担心隐私——持仓数据上传到别人的服务器

所以我决定自己做。**用AI辅助开发，纯Kotlin原生实现，零后端依赖，所有数据存在本地。**

## 它叫什么？

**LXL.AICo**（包名 com.tangtang.aico）

一个Android应用，完全离线运行，不需要注册登录，不花一分钱。

## 能做什么？

### 📊 实时行情
覆盖沪深A股，数据源支持新浪、网易、搜狐，自动切换。

### 📈 K线分析
日K线图展示，支持下载1-15年历史数据。用MPAndroidChart引擎渲染，流畅丝滑。

### 🤖 11种策略插件
内置11种统计模型，可以自由组合：
- **ATR通道** — 波动率突破
- **布林带** — 经典均值回归
- **MACD交叉** — 趋势跟踪
- **RSI反转** — 超买超卖
- **均值回归** — 统计学基础
- **Dual Thrust** — 经典突破策略
- **成交量突破** — 量价配合
- **Parkinson波动率** — 高频数据估算
- **分位数区间** — 概率分布
- **已实现波动率** — 历史波动
- **KDJ** — 经典技术指标

### 🔬 回测引擎
这是最硬核的部分——**93种策略组合的网格评估**，自动计算：
- 总收益率
- 夏普比率
- 最大回撤
- 胜率
- 交易成本（佣金、印花税、过户费、滑点）

不是简单的"金叉买死叉卖"，而是多维度参数寻优。

### 💼 持仓管理
本地JSON存储，实时盈亏计算。支持通过SAF导入持仓文件。

### 🔔 价格监控
自定义价格提醒，实时推送通知。

## 技术架构

**完全零后端。** 所有计算在手机上完成。

```
数据获取 → 新浪/网易/搜狐 API
    ↓
策略引擎 → TechnicalIndicators + ChannelStrategy + TrendStrategy
    ↓
回测引擎 → BacktestEngine（纯Kotlin实现）
    ↓
UI → Jetpack Compose + Material3
```

原始Python量化项目已经全部移植为Kotlin，算法逻辑一行一行手工翻译，确保数值一致。

1. 访问上面的Release链接
