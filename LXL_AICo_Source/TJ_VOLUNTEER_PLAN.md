# 天津高考志愿辅助App — 技术方案

## 一、项目定位

**产品名称：** 天津志愿通（TianjinVolunteer）
**包名：** com.tangtang.tjvolunteer
**定位：** 服务天津市高考考生，综合分数、兴趣、地域、就业，辅助志愿填报决策

## 二、核心功能

### 1. 智能推荐（核心）
- 输入：分数、位次、选科（6选3）、意向地域、意向专业、兴趣标签
- 输出：冲/稳/保 三档推荐，每档含学校+专业+录取概率
- 算法：基于天津历年录取位次，结合用户偏好加权排序

### 2. 院校在天津招生数据
- 全国在天津招生的本科院校（约600-800所）
- 天津本地院校重点展示
- 院校详情：基本信息、在天津招生计划、往年录取线、院校排名

### 3. 专业查询
- 按学科门类/专业类筛选
- 专业详情：介绍、就业方向、薪资、考研方向
- 在天津各院校的该专业实力对比

### 4. 天津历年录取数据
- 本科A/B批、专科批，分科目组合的录取分数/位次
- 数据可视化：折线图看趋势

### 5. 志愿表管理
- 创建多个志愿方案
- 拖拽排序，导出分享

### 6. 天津本地就业数据
- 天津主要产业分布
- 各专业在天津的就业情况

## 三、技术架构

### 前端（Android）
- Kotlin + Jetpack Compose + Material3
- 图表：MPAndroidChart
- 导航：Jetpack Navigation

### 数据层
- **本地SQLite数据库**（预置天津数据包）
- 首次启动从assets加载
- 支持在线更新（GitHub Release增量包）

### 推荐算法（纯Kotlin）
```
录取概率 = f(位次差距, 历年波动, 招生计划变化)
综合得分 = w1*录取概率 + w2*院校匹配 + w3*专业匹配 + w4*地域匹配 + w5*就业匹配
```

## 四、数据方案

### 数据来源
1. 天津教育考试院（历年录取分数/位次）— 公开
2. 教育部阳光高考（院校信息、专业目录）— 公开
3. 天津各高校官网（招生计划、章程）— 公开
4. 天津市统计局（就业数据）— 公开

### 数据格式
```
assets/data/
  universities.json       # 在天津招生的院校信息
  majors.json             # 专业目录
  admission_scores.json   # 天津历年录取数据（分批次/科目组合）
  employment.json         # 天津就业统计数据
  tianjin_universities.json # 天津本地院校详情
```

### 数据量估算
- 在天津招生院校：~800所 × 2KB = 1.6MB
- 专业目录：~800个 × 1KB = 0.8MB
- 录取数据：~5万条 × 200B = 10MB（本科A/B+专科，分科目组合）
- 就业数据：~2000条 × 1KB = 2MB
- **总计约15MB**，非常轻量
